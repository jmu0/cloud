
cloud servers
    prints a list of active servers

cloud shares
    prints a list of shares on active servers

cloud mounts
    prints a list of mounts on active servers 

cloud vmlist 
    prints a list of guest vm's on active servers

cloud migrate <vm> <server>
    live migration of <vm> to <server>

cloud migrate all <from_server> <to_server>
    live migration of all vm's from <from_server> to <to_server>

cloud mount <share>
    mounts share on localhost in /mnt/<server>/<share>

cloud share <path>
    creates nfs share from <path> on localhost
